# Guia de Contribuição

Obrigado por querer contribuir com o projeto Biblioteca Digital em Python!

## Como começar
1. Clone o repositório:
   git clone https://github.com/alinea2106-glitch/biblioteca-digital-python

2. Crie uma nova branch para suas alterações:
   git checkout -b minha-alteracao

3. Faça commits claros e objetivos:
   git commit -m "feat: descreva sua mudança"

4. Envie sua branch para o GitHub:
   git push origin minha-alteracao

5. Abra um Pull Request para revisão.

## Boas práticas
- Sempre escreva mensagens de commit descritivas.
- Mantenha o código organizado.
- Adicione testes quando possível.
